## 6.2 Layered Architecture

```
┌─────────────────────────────────────────────────────────────────┐
│                      Presentation Layer                        │
│  ┌─────────────────────────────────────────────────────────┐  │
│  │  - Floating Widget (input, status, success)            │  │

│  └─────────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐

│  ┌─────────────────────────────────────────────────────────┐  │
│  │  - Goal Manager (capture, track, complete)           │  │
│  │  - Action Planner (Gemini integration)                 │  │
│  │  - State Machine (workflow phases)                     │  │
│  │  - Change Detector (page mutation observer)            │  │
│  └─────────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────────┘

                              ▼
┌─────────────────────────────────────────────────────────────────┐
```
