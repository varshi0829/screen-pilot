# 24. Hackathon Demo Scenario

## 24.1 Demo: Create GitHub Issue

This is a complete walkthrough for a 3-minute demo:

```
┌─────────────────────────────────────────────────────────────────┐
│              Hackathon Demo: Create GitHub Issue                   │
│  ┌─────────────────────────────────────────────────────────┐          │
│  │                                                         │          │
│  │  Setup:                                                 │          │
│  │  1. Open github.com in Chrome                           │          │
│  │  2. Navigate to any repository                        │          │
│  │  3. Click ScreenPilot extension icon                 │          │
│  │                                                         │          │
│  │  Step 1: User Input (5 seconds)                       │          │
│  │  ─────────────────────────────────────────────────     │          │
│  │  User types: "Create issue about login bug"           │          │
│  │  User clicks: "Go"                                 │          │
│  │                                                         │          │
│  │  Step 2: AI Analysis (3 seconds)                  │          │
│  │  ─────────────────────────────────────────────────     │          │
│  │  System shows: "Analyzing..."                      │          │
│  │  Gemini returns: {targetText: "Issues"}           │          │
│  │                                                         │          │
│  │  Step 3: First Highlight (5 seconds)              │          │
│  │  ─────────────────────────────────────────────────     │          │
│  │  System highlights: "Issues" tab                   │          │
│  │  System shows: "Click Issues tab"                   │          │
│  │  User clicks: Issues tab                           │          │
│  │                                                         │          │
│  │  Step 4: Page Change (2 seconds)                   │          │
│  │  ─────────────────────────────────────────────────     │          │
│  │  Page navigates to Issues page                       │          │
│  │  System detects: Page changed                        │          │
│  │  System shows: "Checking..."                        │          │
│  │                                                         │          │
│  │  Step 5: Second Action (3 seconds)                 │          │
│  │  ─────────────────────────────────────────────────     │          │
│  │  Gemini returns: {targetText: "New issue"}        │          │
│  │  System highlights: "New issue" button            │          │
│  │  User clicks: New issue button                    │          │
│  │                                                         │          │
│  │  Step 6: Continue (remaining steps)               │          │
│  │  ─────────────────────────────────────────────────     │          │
│  │  - Fill title field                               │          │
│  │  - Fill description field                        │          │
│  │  - Click submit                                │          │
│  │                                                         │          │
│  │  Step 7: Complete (2 seconds)                   │          │
│  │  ─────────────────────────────────────────────────     │          │
│  │  System shows: "Done! Issue created"              │          │
│  │  Widget minimizes                              │          │
│  │                                                         │          │
│  │  Total Demo Time: ~3 minutes                     │          │
│  └─────────────────────────────────────────────────────────┘          │
```
