# 19. MVP Scope

## 19.1 In-Scope (MVP)

| Feature | Priority |
|---------|----------|
| Floating widget with input | Required |
| Goal capture and send to AI | Required |
| Gemini integration | Required |
| DOM-based element finding | Required |
| Visual highlighting | Required |
| Click-through to element | Required |
| Page change detection | Required |
| Workflow state tracking | Required |
| Success/completion state | Required |
| Error handling | Required |

## 19.2 Out-of-Scope (MVP)

| Feature | Reason |
|---------|--------|
| Voice input | Adds complexity, not MVP |
| Auto-click (no user confirmation) | Requires trust, skip for MVP |
| Multi-tab workflows | Too complex |
| Workflow history | Not required |
| User accounts | Not required |
| Chrome Web Store publishing | Manual install for demo |
| Desktop app support | Web only |
| Mobile support | Not targeted |
