# 23. Future Roadmap

## 23.1 Post-MVP Features

| Feature | Description | Complexity |
|--------|-------------|------------|
| **Auto-click mode** | Automatically click without user confirmation | Medium |
| **Voice input** | Speech-to-text for goal input | Medium |
| **Workflow history** | Save and replay past workflows | Medium |
| **Keyboard shortcuts** | Navigate without mouse | Low |
| **Multi-tab support** | Handle workflows across tabs | High |
| **Custom selectors** | User-defined element selectors | Medium |
| **Site templates** | Pre-built workflows for common sites | Medium |
